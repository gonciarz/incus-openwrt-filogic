incus-openwrt-filogic

Static Incus (client and server) build for OpenWrt/aarch64 (Filogic).

How to add the repository to OpenWrt

wget -O /etc/apk/keys/incus-openwrt-filogic.rsa.pub \
    https://gonciarz.github.io/incus-openwrt-filogic/incus-openwrt-filogic.rsa.pub
echo 'https://gonciarz.github.io/incus-openwrt-filogic/aarch64' > /etc/apk/repositories.d/incus.lst

apk update
apk add attr-static squashfs-tools-unsquashfs-static xz-static incus-static
